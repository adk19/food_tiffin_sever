# How to Convert HTML Files to Word and PowerPoint

I've created HTML versions of your report and presentation that can be easily converted to Word and PowerPoint formats.

## Files Created:

1. **Online_Tiffin_Service_Report.html** - Complete project report
2. **Tiffin_Service_Presentation.html** - PowerPoint presentation (15 slides)
3. **PROJECT_REPORT.md** - Original markdown report

## Method 1: Using Microsoft Word (Recommended)

### For Word Document:
1. Open **Microsoft Word**
2. Go to **File > Open**
3. Select **Online_Tiffin_Service_Report.html**
4. Word will automatically convert the HTML to a Word document
5. Go to **File > Save As** and choose **Word Document (.docx)**
6. Save as "Online_Tiffin_Service_Report.docx"

### For PowerPoint:
1. Open **Microsoft PowerPoint**
2. Go to **File > Open**
3. Select **Tiffin_Service_Presentation.html**
4. PowerPoint will convert it to slides
5. Go to **File > Save As** and choose **PowerPoint Presentation (.pptx)**
6. Save as "Tiffin_Service_Presentation.pptx"

## Method 2: Using Google Docs/Slides (Alternative)

### For Word Document:
1. Open **Google Docs**
2. Go to **File > Import**
3. Upload **Online_Tiffin_Service_Report.html**
4. After import, go to **File > Download > Microsoft Word (.docx)**

### For PowerPoint:
1. Open **Google Slides**
2. Go to **File > Import slides**
3. Upload **Tiffin_Service_Presentation.html**
4. After import, go to **File > Download > Microsoft PowerPoint (.pptx)**

## Method 3: Copy-Paste Method

### For Word:
1. Open **Online_Tiffin_Service_Report.html** in any web browser
2. Select All (Ctrl+A) and Copy (Ctrl+C)
3. Open Microsoft Word
4. Paste (Ctrl+V)
5. Format as needed and save as .docx

### For PowerPoint:
1. Open **Tiffin_Service_Presentation.html** in web browser
2. Each slide is separated - copy each slide content
3. Create new PowerPoint presentation
4. Paste content into individual slides
5. Apply design themes as needed

## Tips for Better Formatting:

1. **For Word Document:**
   - Apply consistent heading styles
   - Add page numbers
   - Include table of contents
   - Use proper margins (1 inch all around)

2. **For PowerPoint:**
